- Check Downloads folder
- Look in File Manager → Downloads
- Search for "*.apk"

### **"Install blocked"**
- Settings → Security → Enable "Install from Unknown Sources"

- Did you uninstall old app first? ← Most common mistake!
- Did you restart phone?
- Try clearing app cache: Settings → Apps → SimpleReactNativeApp → Storage → Clear Cache
